===============================================================================
                            PROJECT OVERVIEW
===============================================================================
Key Features and Flow

--Category & Brand Structure
N-level category structure is implemented for organizing products.
Brands and products are associated with categories for better filtering and navigation.

--Product Customization
When a user purchases a product, they can customize it by changing the text, font family, and text color that will appear on the nameplate.
The customization details (e.g., text, font, color) are stored and included in the order details.

--Options Structure (Similar to OpenCart)
Each product can have generic quantity plus custom options (e.g., red color or specific size).
When an admin disables an option, the system falls back to using the actual product quantity.
Options affect price, weight, and other attributes dynamically.

--Order and Email Notifications
After successful purchase, emails are sent to both the customer and admin, including full details of the customized product (text, font, etc.).

--Media Management
All product images are stored in an Amazon S3 bucket for scalability and fast content delivery.

--Payment Gateway
Stripe payment integration ensures secure and smooth payment processing.

--Filtering and Search
The filter page utilizes the maximum product price for range-based filtering.
Product sizes are fetched and filtered from the options data.

===============================================================================
                            INSTALLATION REQUIREMENTS
===============================================================================

SERVER REQUIREMENTS:
- PHP >= 8.1
- MySQL >= 5.7
- Composer
- Node.js (for asset compilation)
- Web server (Apache/Nginx)

ENVIRONMENT SETUP:
- Configure .env file with database credentials
- Set up AWS S3 bucket and credentials
- Configure Stripe and Flutterwave API keys
- Set up SMTP for email notifications

INSTALLATION STEPS:
1. Clone repository
2. Run: composer install
3. Copy .env.example to .env
4. Configure database and services in .env
5. Run: php artisan key:generate
6. Run: php artisan migrate
7. Run: php artisan db:seed
8. Run: php artisan serve
